# Predictive Analytics Using Historical Data

## Objective
Build a predictive model using historical sales data to forecast future trends.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Dataset
Historical sales data for 6 months.

## Model
Linear Regression

## Output
Predict future sales for upcoming months and visualize trends.
